This is a vrml exporter for 3ds Max version 2020.

If you are having issues with this exporter then please send a mail to:
woessner@hlrs.de


Troubleshooting

During 3ds Max startup, you might get an error message indicating that vrmlexp.dle could not be loaded.
If this is the case, then please install the VisualStudio runtime manually. You can find the installer in:
...Program Files\VrmlExp\vc_redist.x64.exe
